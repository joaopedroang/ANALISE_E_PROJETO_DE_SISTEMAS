# Exercício 02 - Classe TextoSaída

## Cenário
Para fixação do conceito de classes, criação da classe TextoSaída proposta pelo Prof. Ricardo Roberto, focando em atributos de estilo e seleção de componentes.

## 1. Identificação de Classes, Atributos e Métodos
* **Classe:** TextoSaida
* **Atributos:** tamanhoFonte, corFonte, corFundo, tipoComponente, texto.
* **Métodos:** setTamanho(), setCores(), setComponente(), exibir().

## 2. Requisitos Funcionais
* **[RF01]** Configurar atributos de formatação de texto.
* **[RF02]** Escolher tipo de componente de exibição.
* **[RF03]** Validar cores conforme tons pré-definidos.
* **[RF04]** Processar e exibir o texto formatado.

## 3. Requisitos Não Funcionais
* **[RNF01]** Ausência de herança de classes visuais externas.
* **[RNF02]** Implementação da interface em ambiente Web (Streamlit).
* **[RNF03]** Restrição de domínio para cores e componentes específicos.
