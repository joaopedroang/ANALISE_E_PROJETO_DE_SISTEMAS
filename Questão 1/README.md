# Exercício 01 - Acompanhamento de Gasto de Luz

## Cenário
Análise da planilha de controle de gastos mensais de energia do usuário Gabriel.

## 1. Identificação de Classes, Atributos e Métodos
- **Classe:** `Conta`
- **Atributos:** `dataLeitura`, `numLeitura`, `kwGasto`, `valorPagar`, `dataPagamento`, `mediaConsumo`.
- **Métodos:** `verificarMenorConsumo()`, `verificarMaiorConsumo()`.

## 2. Requisitos Funcionais
- [RF01] Cadastrar contas de luz.
- [RF02] Listar histórico de gastos.
- [RF03] Identificar mês de menor consumo.
- [RF04] Identificar mês de maior consumo.

## 3. Requisitos Não Funcionais
- [RNF01] Persistência de dados em arquivo ou banco de dados.
- [RNF02] Suporte a valores decimais para cálculos financeiros.
